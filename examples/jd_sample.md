# AI Agent 开发实习生（虚构 JD）

> 发布方：云杉智能实验室（虚构）

## 岗位职责

- 协助开发基于大语言模型的任务型 Agent
- 编写和维护模型调用、提示词及工具调用代码
- 整理测试样例，记录失败案例并协助改进输出质量
- 与团队成员共同维护项目文档和演示材料

## 任职要求

- 计算机或相关专业在读
- 熟悉 Python，能够完成 API 调用、JSON 处理和文件读写
- 了解大语言模型、Prompt Engineering 或 Tool Calling 基本概念
- 能使用 Git 和 Markdown 进行基础协作
- 能清楚说明项目中的个人工作、问题与改进思路

## 加分项

- 有 Agent、RAG 或自动化工作流相关课程项目
- 有错误处理、测试或模型输出评测经验
